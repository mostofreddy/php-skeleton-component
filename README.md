PHP Skeleton - Component
========================

Base structure for a PHP Component

Version
=======

__0.1.0__

Install
=======

### Requirements

* PHP 5.3+
* [Composer](http://getcomposer.org)

### Via composer

    composer create-project phpskeleton/component path/to/install

Testing
=======

    php vendor/bin/phpunit -c tests/phpunit.xml
